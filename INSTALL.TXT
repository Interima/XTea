=======================================
Installation guide
=======================================

  1. Go to XTea/build directory
	  [ cd XTea/build ] (Linux)

  2. Run cmake with CMakeLists.txt
	  [ cmake CMakeLists.txt ] (Linux)

  3. Go to XTea/lib
	  [ cd ../lib ] (Linux)

  4. Copy lib folder's file to Mena'an plugins directory
	  [ cp ./* <Mena'an/plugins> ] (Linux)

  5. Restart Mena'an
